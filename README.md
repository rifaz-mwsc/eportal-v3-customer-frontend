# LMS Customer Frontend

Angular Material dashboard for the LMS customer portal, built on Modernize Angular template.

## Quick start

Requirements:
- Node.js 18+ (recommended: 20)
- npm 9+

Install and run locally:
- `npm install`
- `npm start`

The app will serve at http://localhost:4200.

## Scripts
- `npm start` — start dev server
- `npm run build` — production build
- `npm test` — run unit tests (if configured)

## CI
This repo includes a basic GitHub Actions workflow that installs dependencies and builds on every push/PR. See `.github/workflows/ci.yml`.

## Notes
- Angular CLI version: 21.x
- If you encounter install/build issues, delete `node_modules` and try `npm ci`.
